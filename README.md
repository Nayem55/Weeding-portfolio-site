# Wedding Invitation React

Premium wedding portfolio / invitation website built with React and Vite.

## Sections

- Cinematic hero with countdown
- Premium experience highlights
- Couple story
- Wedding event timeline
- Venue showcase
- Wedding party
- Editorial gallery
- Guest guide and registry notes
- Love notes/testimonials
- RSVP form

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
